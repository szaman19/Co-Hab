# Co-Hab

This project is to create an open source software that is a group message client,
shared calendar and announcement board.

## Tech :
  - Server : Node.js
  - Middleware: Express
  - Database: MongoDB
  - Frontend: Angular



## Current Working :

  * API
    - User Set Up Interface
    - Group Dashboard Interface
  * Web View Temples
